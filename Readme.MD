# Project Overview

This project predicts whether a customer is likely to churn using machine learning classification algorithms. The objective is to help businesses identify customers who are at risk of leaving so that appropriate retention strategies can be implemented.

# Business Problem

Customer churn directly impacts business revenue. By identifying customers who are likely to leave, organizations can take proactive actions such as targeted offers, personalized communication, and improved customer support.

# Dataset
Dataset: Customer Churn Dataset
Records: 7,043 customers
Features: Customer demographics, service usage, billing details, contract information, and churn status.
